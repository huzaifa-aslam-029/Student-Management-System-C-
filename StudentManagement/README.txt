Student Management System (WinForms .NET 6) - sample solution
-----------------------------------------------------------
Steps to use:
1. Open this folder in Visual Studio 2022+.
2. Ensure you have the Access Database Engine installed (ACE OLEDB).
3. Place your StudentDB.accdb at the path specified in Config.cs (or update Config.DbPath).
4. Build and run.
Notes:
- This is a starter project with minimal designer code. Open Forms folder and edit UI in designer.
- Provided code uses System.Data.OleDb package (included in csproj).
